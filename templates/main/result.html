A Float
B Boolean
C String
D Object

Explanation:

JavaScript has Number type which represents both integer and floating point numbers, but it doesn't have separate Float type. The other options (Boolean, String, Object) are valid JavaScript data types.

Q12 Incorrect

What will be the output of: console.log(1 + "1")?

A 2
B "11"
C NaN
D TypeError

Explanation:

When using the + operator with a number and a string, JavaScript performs string concatenation. The number 1 is coerced into a string "1", resulting in "11".
